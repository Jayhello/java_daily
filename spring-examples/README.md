# Spring Examples

本模块包含Spring框架相关的示例代码。

## 依赖

- Spring Context
- Spring AOP
- Spring JDBC
- Spring Test (测试用)

## 示例

### SpringIoCExample
演示Spring IoC容器和依赖注入的基本用法。

运行示例:
```bash
mvn exec:java -Dexec.mainClass="com.jayhello.spring.SpringIoCExample"
```

## 更多示例

可以在这个模块中添加更多Spring相关的示例，例如:
- AOP切面编程
- Bean生命周期
- 事件发布与监听
- Spring JDBC Template
- 配置管理
