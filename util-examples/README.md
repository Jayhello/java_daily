# Util Examples

本模块包含工具类和模板编程相关的示例代码。

## 依赖

- Apache Commons Lang3
- Apache Commons Collections4
- Google Guava
- Freemarker (模板引擎)
- Velocity (模板引擎)

## 示例

### CommonUtilsExample
演示常用工具类的使用，包括字符串工具和集合工具。

运行示例:
```bash
mvn exec:java -Dexec.mainClass="com.jayhello.util.CommonUtilsExample"
```

### FreemarkerTemplateExample
演示Freemarker模板引擎的基本用法。

运行示例:
```bash
mvn exec:java -Dexec.mainClass="com.jayhello.util.FreemarkerTemplateExample"
```

## 更多示例

可以在这个模块中添加更多工具和模板相关的示例，例如:
- Velocity模板引擎
- JSON处理 (Jackson, Gson)
- XML处理
- 日期时间工具
- 文件操作工具
- 反射工具
- 代码生成器
